预算 = 单价×批量 + 物流费 + 关税
02
中国供应商筛选与评估
通过Alibaba、Made-in-China及LinkedIn筛选12家潜在生产商，采用加权评分模型综合比选，准备推荐报告。
综合评分 = 品质×0.3 + 交期×0.3 + 价格×0.2 + 认证×0.2
03
Incoterms 计算与文件准备
按CIF条款测算完整到岸价，确定最优批次规模，谈定价格、MOQ与交货期，筹备全套出口文件与证书。
CIF = FOB价 + 保险费 + 运费
04
合同执行与物流协调
协调合同条款谈判（含责任、罚款与保险条款），全程监控生产、质检、物流节点直至顺利交付。

工厂审核流程 · FACTORY AUDIT SYSTEM

供应商验证流程

系统化工厂核查 · 降低供应风险 · 区分真实工厂与贸易中间商

8/8
验证项 · 全部通过
针对每一位进入采购流程的供应商，我均会执行以下标准化审核，以区分真实工厂与贸易公司，降低供应链断裂、质量不合格及证书造假的风险。
✓
营业执照与出口资质已验证
核查营业执照有效性及出口权限，确认工厂具备对外贸易主体资格。
✓
生产设备与产能核实已验证
现场或视频确认机器设备与生产线实际状态，与申报产能进行比对核验。
✓
ISO / CE 国际认证已验证
索取并核验ISO、CE等质量证书的真实性及有效期，防范证书造假风险。
✓
实际生产面积核查已验证
通过卫星图像及工厂照片，核实厂区面积与申报数据的一致性。
✓
出口经验确认已验证
通过Panjiva/ImportGenius核查历史出口记录，验证工厂是否具备真实的国际客户服务经验。
✓
样品与定制化能力已验证
评估工厂提供样品及按需定制的能力——区分真实制造商与贸易中间商的核心指标。
✓
供货历史与口碑调查已验证
收集既往买家评价，分析平台信用数据，形成供应商信用档案。
✓
包装与标签符合性已验证
确认工厂包装与产品标签完全符合俄罗斯/CIS市场监管要求与客户规范。
为什么要进行工厂审核？ 中国市场上贸易公司往往将自己包装为制造商，影响价格竞争力、产品质量及供货保障。系统化工厂验证是降低供应链风险、帮助客户找到真正工厂直供资源的核心环节。
